# Savhub 文档

Savhub 是一个基于项目特征的 AI Skill 管理器。它通过预置和自定义的**选择器（Selectors）**分析你的项目特征（文件、框架、语言等），自动从注册表安装匹配的 AI 技能到你的编程客户端（Claude Code、Codex、Cursor 等）。
